### 3. [Task 1] 장바구니 분석 (Market Basket Analysis)
- 알고리즘: Apriori / FP-Growth 활용
- 주요 지표: Support, Confidence, Lift 분석
- 인사이트 추출:
    - 의외의 연관 구매 상품군 발견 (예: 아이스크림 & 특정 스낵)
    - 구매 패턴에 따른 타겟 고객 세그먼트 분류
- 전략 제안: 연관 상품 근접 배치 및 묶음 상품(Set Menu) 구성안


- 가설설정
    - 본 분석의 목적은 고객의 구매 심리를 파악하는 것
- 가설 A (상호 보완 관계): "메인 아이스크림(콘/바)을 구매하는 고객은 입가심용 젤리나 캔디류를 함께 구매할 확률이 높을 것이다."
    - 검증 결과 활용: 계산대 바로 앞(골든 존)에 젤리류 배치 제안.
- 가설 B (가족 단위 구매): "주말 저녁 시간대에는 단품보다는 묶음 상품(멀티팩)이나 대용량 제품의 구매 비중이 높을 것이다."
    - 검증 결과 활용: 주말 전용 패밀리 세트 구성 및 매장 안쪽 대형 냉장고 시인성 강화.
- 가설 C (미끼 상품 효과): "판매 가격이 200원~500원인 초저가 상품은 단독 구매보다 다른 고단가 상품과 함께 구매되는 '미끼' 역할을 수행할 것이다."
    - 검증 결과 활용: 저가 상품의 마진이 적더라도 구색을 갖춰야 하는 근거로 활용.



### 4. [Task 2] 매출 예측 및 날씨 영향도 분석
- 데이터 결합: ASOS(종관기상관측) 기온, 강수량 데이터 매칭
- Feature Engineering :
    - Lag Features: 전월, 전주, 전년 동기 매출액
    - Window Features: 최근 7일 이동 평균 매출
    - Weather Features: 폭염/한파 여부, 불쾌지수 등 파생 변수
- 모델링: LightGBM 기반의 시계열 회귀 모델 구현
- 성능 평가: RMSE, MAE를 통한 모델 검증 및 실제 매출과의 오차 분석


- 가설설정
    - 본 분석의 목적은 외부 환경에 따른 매출 변동성 파악하는 것
- 가설 D (기온 임계점): "최고 기온이 28도를 넘어서는 시점부터 매출 상승 기울기가 급격해지는 '티핑 포인트'가 존재할 것이다."
    - 분석 포인트: 기온과 매출의 상관관계 그래프에서 변곡점 찾기.
- 가설 E (강수량의 역설): "비가 오는 날에는 내방객은 줄어들지만, 한 번 방문할 때 대량으로 구매하는 '사재기' 경향 때문에 객단가는 평소보다 높을 것이다."
    - 검증 결과 활용: 비 오는 날 매장 운영 인력은 줄이되, 배달 서비스 프로모션 강화.
- 가설 F (시차 효과): "오늘의 매출은 어제의 매출(Lag 1)보다, 작년 이맘때(Lag 365) 또는 지난주 같은 요일(Lag 7)의 매출과 더 강한 상관관계를 보일 것이다."
    - 분석 포인트: 시계열 모델의 피처 중요도(Feature Importance) 확인.
